# Instagram User Analysis 📊

## 📌 Project Overview
This project analyzes Instagram data to understand user engagement, post performance, and trends.

## 🎯 Objectives
- Analyze likes and comments
- Identify top-performing posts
- Understand engagement trends

## 🛠️ Tools Used
- Power BI
- Excel

## 📊 Key Insights
- Most engaging posts
- Best time to post
- Engagement patterns
